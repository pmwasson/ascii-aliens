# ascii-aliens
Text based shoot-em-up

## Sketches
Player can bank right or left:                 
```                    
    O   O    O O    
    |\^/|  o/| |\o  
    |>-<|  |<| |>|  
    |/^\|  o\| |/o  
    O   O    O O    
```

Rotating Alien pod:
```
      _             
     / \            
    [|| ]           
     \_/            
```                    

Power-ups in diamonds
```
    /\
   <L+>
    \/
```
